# イベント写真投稿｜汎用運用マニュアル（TGS/レース/展示会）

## 議論層
- **実務層**：投稿の「表記ゆれ」を潰し、量産できる運用にする
- **戦略層**：検索性・タグ経由の流入・公式/出演者への到達率を最大化する
- **創造層**：一言コメントで“現場の空気”を載せ、写真の強度を上げる

---

## 目的
- 1枚ごとに **「イベント / 日付・会場 / ブース(or チーム) / 人物 / 公式アカ / ハッシュタグ」** を固定フォーマットで揃える
- 迷う箇所（表記、敬称、英日併記、公式アカ等）を **ルール化して即コピペ** できる状態にする

---

## 0. 入力情報（毎枚）
最低限この5つが揃えば投稿できる。
1. **イベント名**（例：TOKYO GAME SHOW 2025）
2. **開催日・会場**（例：2025.09.25 – 28｜幕張メッセ）
3. **枠**（TGS=ブース、レース=チーム/サーキット等）
4. **人物名 + アカウント**（モデル/コス/コンパニオン等）
5. **ハッシュタグ**（そのイベントの最小セット）

追加があれば強い
- 企業/作品名、公式アカ、キャラ名、機材/レンズ、撮影位置（屋内/屋外、昼/夜）

---

## 1. 投稿フォーマット（標準テンプレ）
### A. 1枚・人物1名（基本形）
```
📸 {EVENT_EN} – {EVENT_JP}
{DATE}｜{VENUE}

◼︎ {CATEGORY}
{SUBJECT_NAME}（@official_account）

◼︎ {PERSON_ROLE}
{PERSON_NAME} さん
@person_account

{ONE_LINE_COMMENT}

{HASHTAGS}
```

- **{CATEGORY}**：TGSなら「ブース」、レースなら「チーム/ブース/ブランド」、展示会なら「ブース/出展」
- **{SUBJECT_NAME}**：企業名/作品名/チーム名
- **{PERSON_ROLE}**：モデル / コスプレイヤー / イベントコンパニオン / RQ / アンバサダー など
- **{ONE_LINE_COMMENT}**：1行だけ（主観は最小、現場適合の誉めを置く）

### B. 1枚・人物2名（左右など）
```
📸 {EVENT_EN} – {EVENT_JP}
{DATE}｜{VENUE}

◼︎ {CATEGORY}
{SUBJECT_NAME}（@official_account）

◼︎ {PERSON_ROLE}
左：{PERSON1_NAME} さん（@person1_account）
右：{PERSON2_NAME} さん（@person2_account）

{ONE_LINE_COMMENT}

{HASHTAGS}
```

### C. 人物名不明（調査中の運用）
```
📸 {EVENT_EN} – {EVENT_JP}
{DATE}｜{VENUE}

◼︎ {CATEGORY}
{SUBJECT_NAME}（@official_account）

◼︎ {PERSON_ROLE}
※お名前調査中

{ONE_LINE_COMMENT}

{HASHTAGS}
```

## 2. 表記ルール（このスレッドの運用を一般化）
### 2-1. イベント名
- 英日併記を基本：
  - `TOKYO GAME SHOW 2025 – 東京ゲームショウ2025`
- 記号は **「–」**（ハイフンより見た目が整う）

### 2-2. 日付・会場
- 原則：`YYYY.MM.DD – DD｜会場`
- 例：`2025.09.25 – 28｜幕張メッセ`
- 年月日が1日だけなら：`YYYY.MM.DD｜会場`

### 2-3. 区分（◼︎ の見出し）
- TGS：`◼︎ ブース`
- レース：`◼︎ チーム` / `◼︎ サーキット`（必要なら追加）
- 展示会：`◼︎ 出展` / `◼︎ ブース`

### 2-4. アカウント表記
- 公式アカがある場合：`企業名/作品名（@official）`
- 公式アカが無い/不明：
  - `企業名/作品名` のみ（無理に推定しない）
- 人物アカは **単独行** で置く：
  - `@person_account`

### 2-5. 敬称
- 人物は原則 **「さん」** を付与
- ハンドルネームのみでも `さん` を付ける（例：`Ely さん`）

### 2-6. カッコの使い方
- 公式：全角括弧は避け、丸括弧で統一
  - `スマイルゲート（Smilegate）`
- Instagram等URLは、可能なら **@が優先**。URLは補助。

---

## 3. ハッシュタグ運用（最小セット→拡張）
### 3-1. 最小セット（毎回固定）
- イベント固有：例 `#TGS2025`
- 役割カテゴリ：例 `#イベントコンパニオン` / `#RQ` / `#cosplay`

### 3-2. 拡張セット（任意）
- 作品/ゲーム名：`#七つの大罪Origin` など（表記揺れ注意）
- 撮影ジャンル：`#ポートレート` / `#portrait`（イベント側の文化に合わせる）

**ルール**
- まずは **2つ固定**（イベント＋カテゴリ）
- 伸ばしたいときだけ足す（増やしすぎない）

---

## 4. 一言コメント（設計）
- 目的：写真を“現場の文脈”に接続し、無機質な羅列を避ける
- 文字量：**1行**（絵が主）

テンプレ例（イベント向け）
- `爽やかな笑顔がブースの雰囲気にぴったりでした✨`
- `衣装の質感がライトに映えて、めちゃくちゃ強かった。`
- `視線のキレが“作品”の空気をそのまま持ってきてた。`

NG
- 説明過多（3行以上）
- 推測で断定（キャラ名/所属など曖昧なものを言い切る）

---

## 5. 調査フロー（人物/キャラ/所属が不明なとき）
### 5-1. まず現場情報を優先
- ブース掲示 / 配布物 / 名札 / 公式パネルの文字

### 5-2. SNS検索の順序
1) 企業公式アカの投稿（イベント当日）
2) ブース名 + 「コンパニオン」「コスプレ」
3) 写真の衣装/小物で作品名→公式→キャラ

### 5-3. 投稿は止めない
- 不明なら **「※調査中」** で先に出す
- 判明したら、次の投稿で正規表記にする（または固定コメントで追記）
