# 🚀 開発ツールの使い方

## 起動方法（超簡単！）

### 方法1: ダブルクリック（一番簡単）
1. エクスプローラーで `dev.bat` を見つける
2. **ダブルクリック**する
3. メニューが表示されます！

### 方法2: コマンドプロンプトから
1. このフォルダを開く
2. アドレスバーに `cmd` と入力してEnter
3. `dev.bat` と入力してEnter

---

## メニューの使い方

```
╔═══════════════════════════════════════════════╗
║  1. 🚀 アプリを起動     ← 開発サーバー起動    ║
║  2. 📦 セットアップ     ← 初回のみ必要        ║
║  3. 🔍 環境チェック     ← 問題確認            ║
║  4. 🏗️ ビルド           ← 本番用              ║
║  5. 🧪 テスト           ← テスト実行          ║
║  6. 🧹 クリーンアップ   ← 削除                ║
║  7. 🔄 リセット         ← 完全やり直し        ║
║  0. ❌ 終了                                   ║
╚═══════════════════════════════════════════════╝
```

**番号を入力してEnterを押すだけ！**

---

## 初めての場合

1. `dev.bat` をダブルクリック
2. **2** を入力して Enter（セットアップ）
3. 完了したら **1** を入力して Enter（起動）
4. ブラウザで http://localhost:3000 を開く

---

## よくある質問

### Q: 画面が文字化けする
→ Windows ターミナルまたは最新のコマンドプロンプトを使用してください

### Q: Node.jsがないと言われる
→ https://nodejs.org/ からLTS版をダウンロード・インストール
