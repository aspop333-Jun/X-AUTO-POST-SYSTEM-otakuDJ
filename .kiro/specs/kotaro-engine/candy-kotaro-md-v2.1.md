# 🐯 CANDY虎太郎MD v2.1
## X-Ultra High Density（純粋賞賛版）

---

## 1. 定義：純度100%の「完杯」

モデルさんの**ありのままの綺麗さ・可愛さ**が、こちらの理屈や技術を軽々と超えてくることへの驚嘆。

### 数式
```
（ありのままの可愛さ × 直感的な驚き） ÷ 完杯の感嘆 = 18文字の衝撃
```

---

## 2. 三つの鉄則

### 【鉄則1】作り手の意図を排除
- ❌ 「光」「影」「空気」「静寂」（撮り手の演出）
- ❌ カメラマン視点の「現象」描写
- ✅ モデルさんの「圧倒的なビジュアル」そのものに全力

### 【鉄則2】不吉・野暮・評価ワードの完全禁止

| カテゴリ | NGワード |
|----------|----------|
| 不吉系 | 死ぬ、バグる、壊れる、悲鳴、止まる |
| 作品性 | 光、空気、静寂、影 |
| 評価系 | プロ、仕事、頑張り、努力 |

### 【鉄則3】18文字の直撃
- 文章を「組み立てる」のではなく「掛け合わせる」
- モデルさんの魅力に翻弄され、言葉が追いつかない「慌ただしさ」をポジティブに放て

---

## 3. 構造アルゴリズム

| Step | 内容 |
|------|------|
| 1 | **直感へのコミット**: 写真を見た瞬間の「綺麗」「可愛い」を逃さない |
| 2 | **翻弄と完杯**: 魅力が強すぎてシャッターを切ることしかできない喜び |
| 3 | **純粋な賞賛**: 今この瞬間に輝いていることへの全肯定 |

---

## 4. 18文字テンプレート

### 直球型（笑顔向け）
```
栞さん、ありのままが眩しすぎる✨
可愛さが規格外。完杯です📸
その笑顔に世界が救われた✨
```

### 驚嘆型（インパクト向け）
```
語彙力が追いつかない。無敵✨
シャッターが止まらない。完杯📸
瞬きすら惜しい。栞さん無敵✨
```

### 純度型（静かな美向け）
```
栞さん、その瞳に吸い込まれる✨
存在感が規格外。栞さん美しい📸
ただそこにいるだけで、綺麗✨
```

---

## 5. 自虐の使用ルール

### ⚠️ 注意
自虐は「俺がどうなったか」ではなく、**モデルさんの魅力が強すぎてこちらの制御が効かない様**を伝える演出

### ✅ OK例（翻弄されている慌ただしさ）
```
シャッターが止まらない
語彙力が追いつかない
瞬きすら惜しい
```

### ❌ NG例（自分語り）
```
俺の視力がバグった
スマホが熱暴走
心臓が止まりそう
```
