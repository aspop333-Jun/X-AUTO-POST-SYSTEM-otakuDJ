# Kotaro Comment Scoring System
## 4連単 設定値＋同点処理（統合版 / v1.2）

---

## 0. 本MDの位置づけ
本ドキュメントは、**12思考パターンの4連単設定値**と、
**同点が発生した場合の処理ルール**を一体化した最終仕様である。

- 思想説明は行わない
- 実装・比較・公開を前提とする

---

## 1. プライオリティ要素（固定）

| 記号 | 要素名 | 定義 |
|---|---|---|
| A | 表情の確定遅延 | 表情が完全に決まり切らず、余韻が残っている状態 |
| B | 視線の意図未決定 | 視線の向き・意味が断定できない状態 |
| C | 顔パーツ感情非同期 | 目・口・眉などが異なる感情を語っている状態 |
| D | 緊張と緩和の同時存在 | 身体・状況・表情に緊張と抜けが同時に存在 |
| E | 役割と個人の境界揺れ | 衣装・職業・設定と個人の素が混線している状態 |

---

## 2. 12思考パターン × 4連単（確定）

> 4連単 = 最優先 → 第2 → 第3 → 第4
> 
> **沈黙要素** = 高得点でも主判定に使用しない要素

| ID | パターン名 | 4連単（優先順） | 沈黙 |
|---|---|---|---|
| P01 | 直球余韻型 | A → D → B → C | E |
| P02 | 照れ崩し型 | C → B → A → D | E |
| P03 | 無防備回収型 | D → C → A → B | E |
| P04 | 距離錯覚型 | B → A → D → C | E |
| P05 | 静寂侵入型 | D → A → C → B | E |
| P06 | 余白煽動型 | A → B → D → C | E |
| P07 | 日常混入型 | E → D → C → A | B |
| P08 | 理性残留型 | D → B → A → C | E |
| P09 | 視線誘導型 | B → C → D → A | E |
| P10 | 未完成美型 | A → C → D → B | E |
| P11 | 関係未定義型 | E → B → A → D | C |
| P12 | 偶然美化型 | E → C → D → A | B |

---

## 3. 基本判定フロー

1. 画像1枚につき A〜E を **0〜5点**で採点
2. 点数を降順に並べる
3. 上位4要素を抽出し **4連単候補**とする
4. 上位4連単が最も一致するパターンを仮決定

---

## 4. 同点処理ルール（必須）

### 4.1 同点発生条件

- A〜Eの中で **同点が発生**し、
- 複数パターンが競合した場合

---

### 4.2 同点処理ステップ

#### Step 1｜第2要素優先ルール
- 首位要素が同点の場合、
  **第2要素の点数が高いパターンを優先**

---

#### Step 2｜沈黙要素除外
- 競合パターンのうち、
  **沈黙要素が上位4連単に含まれるものは除外**

---

#### Step 3｜E吸引抑制ルール

- 条件：
  - E ≥ 4
  - かつ A ≥ 3 または C ≥ 3

- 処理：
  - P07（日常混入型）を一時除外
  - 残りのパターンで再照合

---

#### Step 4｜最終決定軸

- 上記すべてを適用しても決まらない場合：
  **A（表情の確定遅延）が最も高いパターンを採用**

理由：
- 本システムの最終判断軸は、
  *「表情が確定しきらない瞬間を拾えているか」* にあるため

---

## 5. 出力フォーマット（固定）

```
Image_ID:
A= / B= / C= / D= / E=
→ 上位4連単：◯ → ◯ → ◯ → ◯
→ 判定パターン：PXX｜◯◯型
```

---

## 6. 運用上の注意

- 高スコア＝良い写真ではない
- 本システムは **コメントの攻め方を決める分類器** である
- 同一パターンの連続出現は *失敗ではなく検知結果*
